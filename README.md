# Exercícios em C

Repositório com exercícios desenvolvidos durante meus estudos na linguagem C.

## Estrutura do Projeto

- **01_area_retangulo.c**  
  Programa que calcula a área de um retângulo a partir da base e altura informadas pelo usuário.

- **02_ano_bissexto.c**  
  Verifica se um determinado ano informado pelo usuário é bissexto.

- **03_numero_primo.c**  
  Verifica se um número informado é primo.

- **04_fatorial.c**  
  Calcula o fatorial de um número inteiro positivo.
